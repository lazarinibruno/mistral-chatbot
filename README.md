# Mistral Chatbot

Chatbot developed using Next.js and Mistral AI's chat completion API.

Deployment: [Live Demo](https://mistral-chatbot-five.vercel.app/)

## Features

- Multiple chat support
- Local storage

## Run with Docker

To run the app locally in dev mode, clone the repository and from the project directory run:

```
docker compose -f compose.yaml up --build
```
